# MTProto Proxy Telegram Parser

Автоматический парсер последнего поста из Telegram канала @ProxyMTProto с функцией подключения прокси.

## 🔧 Как это работает

1. GitHub Actions запускается каждые 15 минут
2. Парсит последний пост с https://t.me/s/ProxyMTProto
3. Обновляет index.html и latest_post.json
4. Сайт доступен через GitHub Pages

## 🚀 Функции

- ✅ Автоматическое обновление каждые 15 минут
- ✅ Отображение последнего прокси
- ✅ Кнопка "Подключиться в Telegram" с подстановкой данных
- ✅ Копирование данных в буфер при ошибке
- ✅ Работает в РФ (сервера GitHub за границей)

## 📦 Установка

1. Форкните этот репозиторий
2. Включите GitHub Pages в Settings → Pages
3. Готово! Сайт доступен по адресу: `https://[username].github.io/ProxyMTProto-Parser`

## 📱 Использование

1. Откройте сайт
2. Нажмите "Подключиться в Telegram"
3. Telegram автоматически откроется с подставленными данными прокси

## ⚙️ Технологии

- Python + BeautifulSoup для парсинга
- GitHub Actions для автоматизации
- GitHub Pages для хостинга
