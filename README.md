# TGOnline — рабочие MTProto прокси для Telegram

Автоматический парсер публичного Telegram-канала [@ProxyMTProto](https://t.me/s/ProxyMTProto).  
Сайт обновляется каждые 3 часа и всегда показывает **4 последних рабочих прокси**.

🔗 **Ссылка на сайт:** [https://blog1703.github.io/tgonline/](https://blog1703.github.io/tgonline/)

---

## 📌 О проекте

- 🔄 **Парсинг:** `https://t.me/s/ProxyMTProto`
- 📦 **Хостинг:** GitHub Pages
- ⏱ **Обновление:** каждые 3 часа (через GitHub Actions)
- 🧠 **Технологии:** Python + BeautifulSoup

---

## 🔍 Для чего это нужно

Сайт даёт быстрый доступ к свежим MTProto прокси без необходимости заходить в Telegram.  
Подходит для пользователей из РФ, где Telegram может быть заблокирован.
