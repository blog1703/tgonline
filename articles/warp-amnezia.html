Доступный интернет всем: создаём конфиг Cloudflare WARP для AmneziaVPN бесплатно

Этот bash-скрипт сгенерирует конфиг Cloudflare WARP для AmneziaVPN. Не стоит выполнять его локально, так как РКН заблокировал запросы для получения конфига. Вместо этого лучше выполнять на удалённых серверах или в GitHub Codespaces.

Важно: Не выполняйте скрипт на локальной машине в РФ — из-за блокировок DNS запросы не пройдут. Codespaces (сервера GitHub за границей) идеально подходят для этой задачи.

Что даёт этот метод: Доступ к заблокированным ресурсам — Twitter, Instagram, YouTube, Facebook и другим сайтам, которые недоступны с территории РФ без обхода. Данный метод не обходит «белый список», он работает только для уже заблокированных сайтов.

GitHub Codespaces — пошаговая инструкция

1 Переходим в репозиторий

Откройте ссылку: https://github.com/ImMALWARE/bash-warp-generator/codespaces

2 Создаём Codespace

Введите учётные данные GitHub (потребуется авторизация, если вы не вошли в аккаунт). Нажмите Create codespace on main и дождитесь загрузки среды. Обычно это занимает 10–30 секунд.

Нажмите на изображение, чтобы увеличить — терминал внизу экрана после загрузки Codespaces
3 Запускаем генератор конфига

В терминале (внизу экрана) вводим команду и нажимаем Enter:

bash warp_generator.sh
4 Копируем готовый конфиг

После того, как конфиг сгенерируется, скопируйте его — можно сохранить на рабочем столе в текстовом файле .txt или сразу в буфер обмена. Конфигурация будет представлена в формате, совместимом с AmneziaVPN.

Нажмите на изображение, чтобы увеличить — выделите вывод терминала и скопируйте конфиг
5 Импортируем в AmneziaVPN

Скопированный конфиг импортируем в приложение AmneziaVPN. Откройте AmneziaVPN → нажмите "Добавить VPN" → выберите "Импорт из файла" или "Вставить конфигурацию". Вставьте скопированный текст и сохраните.

Скачать AmneziaVPN: https://wiki.malw.link/network/vpns/amneziavpn

Скачать AmneziaVPN c GitHub вкладка Releases: https://github.com/amnezia-vpn/amnezia-client
AmneziaVPN — это клиент с открытым исходным кодом, поддерживающий различные протоколы. Конфиг, сгенерированный скриптом, работает через протокол WireGuard, который используется Cloudflare WARP.

6 Подключаемся и пользуемся

После импорта конфига нажмите "Подключиться". Если всё сделано правильно, соединение установится, и вы сможете пользоваться быстрым и безопасным интернетом через сеть Cloudflare WARP.

Обязательно удаляем Codespace!

После завершения обязательно удалите созданное виртуальное окружение:

  • Переходим в https://github.com/ImMALWARE/bash-warp-generator/codespaces
  • Находим свой codespace, нажимаем на три точки (···) → Delete
  • GitHub удаляет Codespaces автоматически через некоторое время бездействия, но лучше удалить сразу, чтобы не тратить бесплатные часы.
Нажмите на изображение, чтобы увеличить — меню удаления Codespace: три точки → Delete

Альтернативный способ: AmneziaWG (нативный клиент)

Если вы не хотите использовать AmneziaVPN, есть ещё один вариант — AmneziaWG. Это отдельный клиент, созданный специально для протокола WireGuard с усилением (амнезия-режим). Он легче, быстрее и делает трафик ещё более незаметным для DPI.

Для чего нужен AmneziaWG:

  • Обход глубокого анализа пакетов (DPI) там, где обычный WireGuard уже не работает
  • Минимальное потребление ресурсов
  • Работает как отдельное приложение, не требует установки AmneziaVPN

Как получить конфиг для AmneziaWG

AmneziaWG использует конфигурационный файл в формате .conf. Получить его очень просто:

  1. Откройте любой поисковик (Google, DuckDuckGo)
  2. Введите запрос: warp amneziawg generator
  3. Перейдите по одной из первых ссылок (обычно это страницы на Vercel или GitHub Pages)
  4. Нажмите кнопку «Генерация» или «Скачать конфиг»
  5. Сохраните полученный файл. Имя файла должно содержать только латинские буквы и цифры, например warp_config.conf

Импорт конфига в AmneziaWG

Windows:
Запустите AmneziaWG → нажмите «Импорт конфигурации» → выберите скачанный .conf файл → Подключиться

Android:
Откройте приложение → нажмите на кнопку «+» → выберите «Импорт из файла» → укажите .conf файл → Подключиться

Linux:
Установите модуль ядра и используйте утилиту wg-quick с файлом .conf

После подключения вы получите тот же защищённый канал через Cloudflare WARP, но с более лёгким и специализированным клиентом.

Готово! После импорта конфига в AmneziaVPN или AmneziaWG вы сможете пользоваться быстрым и безопасным интернетом через Cloudflare WARP. Все шаги проверены и работают.
